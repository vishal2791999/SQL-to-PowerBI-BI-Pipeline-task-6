# SQL to Power BI Business Intelligence Project

## Overview
This project demonstrates a complete SQL-to-dashboard pipeline.

## Features
- SQL database with normalized tables
- Advanced SQL queries (CTE, Window Functions)
- Reporting Views
- Stored Procedure for monthly reporting
- Power BI dashboard using DirectQuery

## Tools Used
- SQL Server
- Power BI
- Excel Dataset

## SQL Components
- vw_CustomerKPIs
- vw_MonthlyTrend
- vw_ProductPerformance
- vw_RegionalSummary
- sp_MonthlySummary

## How to Run
1. Execute SQL script
2. Load dataset into tables
3. Connect Power BI to SQL
4. Build dashboard
